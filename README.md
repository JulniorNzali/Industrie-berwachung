# 🏭 Industrielles Überwachungssystem

Python-Projekt zur Überwachung einer industriellen Produktionslinie mit Anomalieerkennung und prädiktivem maschinellem Lernen.

## 🎯 Ziel

Simulation, Analyse und Überwachung von Daten industrieller Sensoren (Temperatur, Vibration, Druck) in Echtzeit.

## 🛠️ Verwendete Technologien

- **Python 3.14**
- **Pandas** — Datenbearbeitung und -analyse
- **Matplotlib** — Datenvisualisierung
- **NumPy / SciPy** — Wissenschaftliche Berechnungen und Statistik
- **Scikit-learn** — Machine Learning (Random Forest)
- **fpdf2** — Automatische Erstellung von PDF-Berichten

## 📊 Funktionen

- ✅ Simulation von 1000 Messungen industrieller Sensoren
- ✅ Statistische Datenanalyse (Min., Max., Mittelwert)
- ✅ Visualisierung mit 4 professionellen Diagrammen
- ✅ Erkennung von Anomalien mittels Z-Score und gleitendem Durchschnitt
- ✅ Prädiktives ML-Modell (100 % Genauigkeit)
- ✅ Automatisches Warnsystem
- ✅ Automatisch generierter PDF-Bericht
